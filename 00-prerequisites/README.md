# 00 - 前置知识：ML/DL、LLM、RL 基础

> 如果你已经有 ML/DL 和 LLM 的基础，可以直接跳到 [01-dcai-foundations](../01-dcai-foundations/)。

## ML/DL 入门

### 通用课程

- [李宏毅机器学习](https://speech.ee.ntu.edu.tw/~hylee/ml/2023-spring.php) -- 入门神课。初学者不要陷入刷课怪圈，跟着看到 20 多集、看完 Transformer 就算有了基本认识，之后的主题需要用到时再去了解
- [动手学深度学习 (d2l)](https://d2l.ai/chapter_introduction/index.html) -- 可以把第 6/10/13 章节快速过一下，有 NLP 需求也快速过一下即可。推荐在 Colab 上跑跑例程

### 经典论文

> 一些必知必看的经典论文和模型，范围比较 general，重要的是介绍思想和积累认识。

- [Attention Is All You Need](https://arxiv.org/abs/1706.03762) -- Transformer 原始论文
- [BERT](https://arxiv.org/abs/1810.04805) -- 预训练语言模型
- [ResNet](https://arxiv.org/abs/1512.03385) -- 残差网络

## LLM 入门

### 公开仓库与教材

- [Stanford CS25: Transformers United](https://web.stanford.edu/class/cs25/) -- 从 Transformer 的古往今来开始，介绍了对齐/MoE 等前沿方向，请到各机构前沿研究人员给 talk
- [Stanford LLM Basic](https://stanford-cs324.github.io/winter2022/)
- [复旦大学 LLM 教材](https://intro-llm.github.io/)
- [人民大学 LLM 综述](https://arxiv.org/abs/2303.18223)

## 强化学习 / RLHF 入门

- [经典 RL 教材: Sutton & Barto](http://incompleteideas.net/book/the-book-2nd.html)
- [复旦大学 LLM 教材 - RLHF 部分](https://intro-llm.github.io/)
- [RLHF 综述: A Survey of Reinforcement Learning from Human Feedback](https://arxiv.org/abs/2404.18922) -- 涵盖 DPO、PPO 及各自优势等

## 学习建议

1. **不要过度刷课**：ML/DL 基础课程了解核心概念即可，更多时间应该花在动手实践上
2. **跑通代码优先**：推荐没跑过神经网络的人先跑通几个经典模型的代码，帮助快速建立认识
3. **带着思考看论文**：看看现在研究在关注什么问题、怎么解决的、还有哪些没做好的地方
